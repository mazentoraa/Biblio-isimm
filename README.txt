# 📚 Gestion Emprunts App

A desktop application dedicated to managing students, books, and book borrowing operations. This lightweight system allows you to:

- Add, edit, delete, search, and display student/book records.
- Handle borrowing operations efficiently.

This version is customized for **The Higher Institute of Informatics and Mathematics of Monastir (ISIMM)**.

## 🖥️ Installation Guide

1. **Right-click** on the `.zip` file named `Gestion Emprunts App`.
2. Choose **“Extract here”** (or **“Extraire ici”**).
3. A new folder with the same name will appear.
4. Open the folder and look for the executable file named `Gestion Emprunt`.
5. **Double-click** the executable to launch the app and enjoy!

> 💡 To place the app on your desktop, right-click on the executable, choose **"Create Shortcut"** (or **"Créer un raccourci"**) and move the shortcut to your desired location.

## 📂 Data Storage

The application uses **3 CSV files** as its database to store:

- Student data
- Book inventory
- Borrowing records
